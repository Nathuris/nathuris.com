这是我的第一个 Vibe Coding 项目——这个个人网站本身。

## 项目特点

- 使用 Astro 静态网站框架构建
- Tailwind CSS 简约设计
- 支持暗色/亮色模式
- 手机和电脑都能完美显示
- 通过 CMS 后台轻松管理内容

通过后台管理系统，你可以添加更多项目作品。
